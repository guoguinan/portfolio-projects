# Portfolio Projects - 高级前端工程师作品集

为远程外企求职/接单准备的5个高质量作品集项目。

## 项目列表

### 1. Portfolio Website
- **技术栈**: Next.js 16 + TypeScript + Tailwind CSS v4 + Framer Motion
- **特点**: 个人作品集网站，包含 Hero、About、Skills、Projects、Experience、Contact 等区块
- **构建状态**: ✅ 成功
- **目录**: `portfolio-website/`

### 2. E-commerce Admin
- **技术栈**: Next.js 16 + TypeScript + Tailwind CSS v4 + Recharts
- **特点**: 电商管理后台，包含仪表盘、商品管理、订单管理、数据可视化
- **构建状态**: ✅ 成功
- **目录**: `ecommerce-admin/`

### 3. Collaborative Whiteboard
- **技术栈**: React 19 + TypeScript + Vite + Tailwind CSS v4 + Zustand + Socket.io
- **特点**: 协作白板应用，支持实时协作、画笔工具、图形绘制
- **构建状态**: ✅ 成功
- **目录**: `collaborative-whiteboard/`

### 4. Social Analytics
- **技术栈**: React 19 + TypeScript + Vite + Tailwind CSS v4 + Recharts
- **特点**: 社交媒体数据分析仪表盘，包含多平台数据聚合、图表可视化
- **构建状态**: ✅ 成功
- **目录**: `social-analytics/`

### 5. Task Management
- **技术栈**: React 19 + TypeScript + Vite + Tailwind CSS v4 + DnD Kit
- **特点**: 任务管理看板，支持拖拽排序、多列状态管理、任务筛选
- **构建状态**: ✅ 成功
- **目录**: `task-management/`

## 快速开始

```bash
# 进入任意项目
cd portfolio-website

# 安装依赖
npm install

# 开发模式
npm run dev

# 生产构建
npm run build
```

## 部署

所有项目已配置好，可以直接部署到 Vercel：

```bash
# 使用部署脚本
./deploy.sh

# 或手动部署（进入每个项目目录）
cd portfolio-website
vercel --prod
```
